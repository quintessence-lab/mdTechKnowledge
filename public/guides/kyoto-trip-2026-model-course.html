4日間の行程

一日目
移動 ＋ 二条城・金閣寺
到着後すぐ洛中へ。初日から世界遺産2件を巡る
エリア「洛中（らくちゅう）」は京都御所を中心とする旧市街の中心部で、二条城もこのエリア。「洛北」は市街地北側の山際で、金閣寺（鹿苑寺）が代表格。京都駅から見ると北西方向へ移動していく1日。
Day1の動線（京都駅 → 二条城 → 金閣寺）
午前東京駅発 のぞみ → 京都着、ホテルに荷物を預ける
昼前二条城（国宝・二の丸御殿）2026年は寛永行幸四百年祭
昼蕎麦たつ市：二条城近くの町家蕎麦。鴨せいろ・鴨なんばんが絶品
午後金閣寺：金色の舎利殿は圧巻。拝観受付は16:30頃まで
夕方ホテルチェックイン。翌朝に備えて早めに休む
教訓人気店の昼食は待ち時間で午後が押しやすい。この日も待ちが長引き、予定していた北野天満宮を断念。昼は開店直後を狙うか、押したら午後の行程を1つ削る判断を。
二日目
東山縦断 ── 清水寺から銀閣寺まで
朝の清水寺に始まり、夜の川床で締める東山フルコース
エリア「東山（ひがしやま）」は鴨川の東側、山際に寺社が連なる京都観光の王道エリア。清水寺・八坂神社（祇園）・南禅寺・銀閣寺までは南北に一直線で、徒歩やバスでつなげる一筆書きルートが組みやすい。
Day2の動線（清水寺 → 八坂神社周辺 → 南禅寺 → 銀閣寺）
07:30清水寺 朝イチ参拝（開門6:00。8時台までは人が少なく涼しい）
09:00二年坂・三年坂を散策
10:00八坂庚申堂（くくり猿）→ 八坂神社
11:00漢字ミュージアム：八坂神社の目の前。冷房の効いた体験型施設で、暑い時間帯の避難先としても優秀
13:00昼食：南禅寺 順正：名物の湯豆腐・引き上げ湯葉。庭園も見事
14:30南禅寺側から哲学の道を北上
夕方銀閣寺（東山慈照寺）
夜バスで祇園へ戻り、鴨川沿いを散歩しながら店探し → 京いざかや くぼ田の川床で夕食：夏の鱧の湯引き・天ぷらが絶品
ポイント「朝の清水寺 → 昼の屋内（漢字ミュージアム・順正）→ 夕方の銀閣寺 → 夜の川床」と、暑さのピークをうまく外した理想的な配分になった。
三日目
伏見・宇治 ── 京阪沿線の一筆書き
千本鳥居から宇治茶の本場へ。帰り道に三十三間堂を前倒し
エリア「伏見」は市街地南部、千本鳥居の伏見稲荷大社が中心。「宇治」はさらに南、京都市街とは別の宇治市で、平等院や宇治茶で有名。「三十三間堂」は京都駅にほど近い東山南端。3エリアがすべて京阪本線でつながっているのがこの日の肝。
Day3の動線（伏見稲荷 → 宇治 → 三十三間堂）
朝伏見稲荷大社 朝イチで千本鳥居へ（混雑と暑さを避けるなら8時前着が理想）
午前参道で土産＆名物：ちいかわもぐもぐ本舗、日野家のスズメの串焼き・稲荷寿司
昼過ぎ京阪で宇治へ → 平等院鳳凰堂
午後増田茶舗で宇治抹茶ラテ → 茶づな（お茶と宇治のまち歴史公園）で涼む※抹茶挽きなどの体験は時間指定制のため、やりたい場合は事前予約推奨
15:40京阪宇治駅から乗車 → 七条駅下車
16:15三十三間堂（受付16:30まで。堂内は日差しがなく夕方でも快適）
夕方京都駅で土産購入 → 夕食
ポイント伏見稲荷・宇治・三十三間堂はすべて京阪沿線。帰り道に七条で途中下車するだけで三十三間堂を消化でき、最終日の自由度が大きく上がった。猛暑日は午後を茶房・ミュージアムなど屋内で組むのが正解。
四日目
嵐山・太秦 ＋ 帰京
朝の竹林から映画村まで。夜のぞみでゆとりの最終日
エリア「嵐山（あらしやま）」は市街地西端、桂川沿いの景勝地で竹林・渡月橋が象徴。隣接する「太秦（うずまさ）」は東映太秦映画村がある撮影所の街。嵐電（京福電鉄）1本でつながっており、京都駅からは一番西に離れたエリア。
Day4の動線（竹林 → 天龍寺 → 渡月橋 → 太秦映画村）
08:00チェックアウト、荷物は京都駅コインロッカーへ
08:45竹林の小径（朝の涼しく空いているうちに）→ そのまま天龍寺 北門から入り曹源池庭園を拝観、正門から出る
昼前渡月橋を渡り、峯嵐堂 渡月橋本店でできたてわらび餅
昼嵯峨とうふ 稲 本店（天龍寺正門前）で湯葉うどん
午後嵐電で東映太秦映画村へ（2026年3月リニューアル。屋内アトラクション多めで午後の暑さ対策にも◎）。たっぷり遊ぶ
夕方京都駅へ → 駅弁を購入
19:00頃のぞみ乗車 → 21:15頃 東京着
ポイント竹林 → 天龍寺北門 → 庭園 → 正門 → 渡月橋は、坂を戻らずに歩ける最効率ルート。嵐山と映画村は嵐電1本でつながる。最終日の新幹線を19時台にしたことで、映画村を駆け足にせず満喫できた。

この旅から得た夏の京都Tips

  1. 「朝イチ屋外・午後屋内」の配分が夏の京都攻略の基本。漢字ミュージアム、茶づな、太秦映画村は冷房の効いた「避難先」としても機能する。
  2. 人気飲食店の待ち時間はスケジュールリスク。開店直後を狙うか、押したら後続の行程を削る覚悟を。
  3. 京阪沿線（伏見稲荷〜宇治〜七条）は一筆書きの黄金ルート。三十三間堂の「帰りに途中下車」は特におすすめ。
  4. 行けなかった場所は翌日以降に無理に押し込まない。今回は北野天満宮・東寺を潔くスキップし、代わりに嵐山を回収して満足度が上がった。
  5. 帰りの新幹線は遅めに設定しておくと最終日の自由度が段違い。駅弁で車内夕食にすれば時間を有効活用できる。
  6. 体験系施設（茶づなの抹茶挽きなど）は時間指定制が多いので事前予約を。
2026年7月実施の実録ベース ── 営業時間・料金・開催日程は変更される場合があるため、訪問前に各公式サイトで最新情報の確認を推奨します。
