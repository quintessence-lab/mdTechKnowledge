Q / K / V の検索エンジンアナロジー

Self-Attentionの3要素を「検索エンジン」に例えて理解する

Query (Q)
検索クエリ
「何を探しているか」
— 各トークンが他のトークンに問い合わせる内容
Key (K)
タグ / 見出し
「何を持っているか」
— 各トークンが公開する"検索用インデックス"
Value (V)
コンテンツ
「実際に渡す情報」
— マッチしたとき返される中身のデータ
具体例: "cat" が Query を発行する場合
The cat sat on the mat
Qcat と各トークンの K の内積 → 類似度スコア

Softmax後のスコアで V を加重平均
出力 = 各トークンの Value をAttentionスコアで加重合計
Outputcat = 0.05·VThe + 0.42·Vcat + 0.28·Vsat + 0.03·Von + 0.04·Vthe + 0.18·Vmat
