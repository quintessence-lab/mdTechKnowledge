## 概要

Anthropicは2026年3月、AIの学習プラットフォーム「**Anthropic Academy**」と公式技術認定資格「**Claude Certified Architect (CCA)**」を発表した。本記事では学習コース・修了証・認定資格の全体像を整理する。

---

## Anthropic Academy

### 基本情報

| 項目 | 内容 |
|------|------|
| **URL** | https://anthropic.skilljar.com/ |
| **開始日** | 2026年3月2日 |
| **費用** | 完全無料（Claudeの有料サブスクリプション不要） |
| **登録** | メールアドレスのみ |
| **修了証** | 各コース完了時にPDF形式で発行（LinkedIn追加可能） |
| **形式** | セルフペース（自分のペースで学習） |

### 3つの学習トラック

| トラック | 対象者 | 内容 |
|----------|--------|------|
| **AI Fluency** | 全般（学生・教育者・マネージャー等） | AIの基礎概念、責任あるAI活用の方法論 |
| **Product Training** | Claudeユーザー | Claude製品の効果的な使い方 |
| **Developer Deep-Dives** | 開発者 | Claude API統合、MCPサーバー構築、Claude Code活用（8時間超のコンテンツ） |

### 提供コース数

2026年3月時点で**13コース**が提供されている。開発者向けコースでは、システムプロンプト設計、ツール使用（Tool Use）、コンテキストウィンドウ管理、アプリケーションアーキテクチャパターンなどを扱う。

### 関連する学習リソース

| リソース | URL | 概要 |
|----------|-----|------|
| **Coursera上のAnthropicコース** | https://www.coursera.org/partners/anthropic | Coursera経由で受講可能なコース群 |
| **Claude Campus Program** | https://claude.com/programs/campus | 大学・高等教育機関向けプログラム |
| **Claude for Education** | https://claude.com/solutions/education | 教育機関向けソリューション全般 |

## 認定資格: Claude Certified Architect (CCA)

### Claude Certified Architect — Foundations (CCA-F)

Anthropic初の公式技術認定資格。2026年3月12日にリリースされた。

| 項目 | 内容 |
|------|------|
| **正式名称** | Claude Certified Architect — Foundations (CCA-F) |
| **リリース日** | 2026年3月12日 |
| **形式** | 試験（パス/フェイル） |
| **費用** | Claude Partner Network会員は先着5,000名無料（通常$99） |
| **対象** | 開発者・アーキテクト |
| **位置づけ** | CCA認定資格シリーズのエントリーポイント |

### 出題ドメイン（5分野）

1. Claude APIの基礎と統合
2. プロンプト設計とエンジニアリング
3. アプリケーションアーキテクチャ
4. セキュリティとベストプラクティス
5. パフォーマンス最適化と運用

### 今後の認定資格ロードマップ（2026年中）

Anthropicは以下の追加認定資格を2026年中にリリース予定と発表している：

- **セールス向け認定** — Claudeの提案・導入支援に関する資格
- **開発者向け認定** — より高度な実装スキルを問う資格
- **アドバンスドアーキテクト** — CCA-Fの上位資格

## 学習の進め方（推奨ステップ）

1. **Anthropic Academy** でAI Fluencyトラックから開始（無料）
2. 開発者であれば **Developer Deep-Dives** トラックを受講
3. 各コースの**修了証**を取得
4. **CCA-F試験**に挑戦（Claude Partner Network加入で無料枠あり）

## 2026年4月リリース対応の最新動向（追記：2026-04-29）

2026年4月にClaude Opus 4.7（4月16日リリース）、Claude Design（4月17日リリース、Anthropic Labs）、およびClaude Mythos Preview（プレビュー提供開始）が発表されました。これに伴うAnthropic Academyおよび認定資格まわりの動向を、公式に確認できる範囲で整理します。

### Anthropic Academyのコース拡充

2026年3月時点で13コースだった提供コース数は、4月時点で**17コース**に拡充されています（Anthropic Academy公式サイト掲載分）。新たに確認できる主なコースは以下のとおりです。

| 新規コース | トラック | 概要 |
|------------|----------|------|
| **Introduction to Claude Cowork** | Product Training | Coworkデスクトップエージェントのタスクループ、プラグイン／スキル、ファイル・リサーチワークフローの基礎 |
| **Introduction to agent skills** | Developer Deep-Dives | エージェントスキル機能の入門 |
| **Introduction to subagents** | Developer Deep-Dives | サブエージェントによる多段階ワークフロー設計 |
| **AI Capabilities and Limitations** | AI Fluency | モデルが実際にできること／できないことの理解 |

> 2026年4月時点で確認できる情報として、Anthropic Academy公式サイト（https://anthropic.skilljar.com/）に上記コースが掲載されています。Claude Opus 4.7・Claude Design・Mythos Preview**専用**の独立コースは、2026年4月時点では公式サイト上で確認できません。

### Claude Certified Architect (CCA-F)のシラバス

2026年4月時点で公開されているCCA-F試験のシラバスは、5つのドメインと配分が以下のように整理されています（記事公開当初のドメイン名から、より具体的な領域名に整理されています）。

| ドメイン | 配分 |
|----------|------|
| **Agentic Architecture & Orchestration**（エージェント型アーキテクチャとオーケストレーション） | 27% |
| **Claude Code Configuration & Workflows**（Claude Code設定とワークフロー） | 20% |
| **Prompt Engineering & Structured Output**（プロンプト設計と構造化出力） | 20% |
| **Tool Design & MCP Integration**（ツール設計とMCP統合） | 18% |
| **Context Management & Reliability**（コンテキスト管理と信頼性） | 15% |

試験形式の補足として、シナリオベースの設問構成（複数のユースケースから一部がランダム出題）であることが、Anthropic Partner Networkおよび試験準備関連の公開情報で確認できます。

> 2026年4月時点で確認できる情報として、Claude Code関連の出題比重（CLAUDE.mdの階層、スキルのfrontmatter、CI/CDフラグなど）や、MCP統合の比重が明示的に拡充されている点が公式ロードマップ・試験ガイドから読み取れます。Claude Opus 4.7・Claude Design・Mythos Preview固有の出題追加に関する公式アナウンスは、2026年4月時点では確認できません。

### 上位認定資格の状況

2026年3月発表の認定ロードマップ（セールス向け／開発者向け／アドバンスドアーキテクト）について、2026年4月時点でCCA-F以外の正式リリースは公式に確認できていません。続報を待つ段階です。
