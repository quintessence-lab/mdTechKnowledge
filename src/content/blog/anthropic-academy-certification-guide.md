## 概要

Anthropicは2026年3月、AIの学習プラットフォーム「**Anthropic Academy**」と公式技術認定資格「**Claude Certified Architect (CCA)**」を発表した。本記事では学習コース・修了証・認定資格の全体像を整理する。

---

## Anthropic Academy

### 基本情報

| 項目 | 内容 |
|------|------|
| **URL** | https://anthropic.skilljar.com/ |
| **開始日** | 2026年3月2日 |
| **費用** | 完全無料（Claudeの有料サブスクリプション不要） |
| **登録** | メールアドレスのみ |
| **修了証** | 各コース完了時にPDF形式で発行（LinkedIn追加可能） |
| **形式** | セルフペース（自分のペースで学習） |

### 3つの学習トラック

| トラック | 対象者 | 内容 |
|----------|--------|------|
| **AI Fluency** | 全般（学生・教育者・マネージャー等） | AIの基礎概念、責任あるAI活用の方法論 |
| **Product Training** | Claudeユーザー | Claude製品の効果的な使い方 |
| **Developer Deep-Dives** | 開発者 | Claude API統合、MCPサーバー構築、Claude Code活用（8時間超のコンテンツ） |

### 提供コース数

2026年3月時点で**13コース**が提供されている。開発者向けコースでは、システムプロンプト設計、ツール使用（Tool Use）、コンテキストウィンドウ管理、アプリケーションアーキテクチャパターンなどを扱う。

### 関連する学習リソース

| リソース | URL | 概要 |
|----------|-----|------|
| **Coursera上のAnthropicコース** | https://www.coursera.org/partners/anthropic | Coursera経由で受講可能なコース群 |
| **Claude Campus Program** | https://claude.com/programs/campus | 大学・高等教育機関向けプログラム |
| **Claude for Education** | https://claude.com/solutions/education | 教育機関向けソリューション全般 |

## 認定資格: Claude Certified Architect (CCA)

### Claude Certified Architect — Foundations (CCA-F)

Anthropic初の公式技術認定資格。2026年3月12日にリリースされた。

| 項目 | 内容 |
|------|------|
| **正式名称** | Claude Certified Architect — Foundations (CCA-F) |
| **リリース日** | 2026年3月12日 |
| **形式** | 試験（パス/フェイル） |
| **費用** | Claude Partner Network会員は先着5,000名無料（通常$99） |
| **対象** | 開発者・アーキテクト |
| **位置づけ** | CCA認定資格シリーズのエントリーポイント |

### 出題ドメイン（5分野）

1. Claude APIの基礎と統合
2. プロンプト設計とエンジニアリング
3. アプリケーションアーキテクチャ
4. セキュリティとベストプラクティス
5. パフォーマンス最適化と運用

### 今後の認定資格ロードマップ（2026年中）

Anthropicは以下の追加認定資格を2026年中にリリース予定と発表している：

- **セールス向け認定** — Claudeの提案・導入支援に関する資格
- **開発者向け認定** — より高度な実装スキルを問う資格
- **アドバンスドアーキテクト** — CCA-Fの上位資格

---

## 学習の進め方（推奨ステップ）

1. **Anthropic Academy** でAI Fluencyトラックから開始（無料）
2. 開発者であれば **Developer Deep-Dives** トラックを受講
3. 各コースの**修了証**を取得
4. **CCA-F試験**に挑戦（Claude Partner Network加入で無料枠あり）
