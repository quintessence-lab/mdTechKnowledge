claude.ai / code.claude.comからClaude Codeを利用する際のアーキテクチャ図。

```
┌──────────────────────────────────────────────────────────────┐
│                    あなたの操作端末                             │
│                                                              │
│  ┌──────────────┐    ┌──────────────────────────────────┐   │
│  │ ローカル      │    │ ブラウザ                          │   │
│  │ ファイル      │    │                                  │   │

│  │              │    │  │  チャットUI                  │  │   │
│  │  ← × →      │    │  │  「リポジトリとは？」 [送信]   │  │   │
│  │  直接アクセス  │    │  │         ↑ 表示  │ 送信       │  │   │
│  │  不可         │    │  └─────────┼──────┼───────────┘  │   │
│  └──────────────┘    └────────────┼──────┼───────────────┘   │
│         │                         │      │                    │
└─────────┼─────────────────────────┼──────┼────────────────────┘
          │ git pull                │      │
          │ (手動)                  │      │ HTTPS (WebSocket)
          │                         │      │
          │    ┌────────────────────┼──────┼────────────────┐
          │    │   Anthropic クラウド │      │                │
          │    │                    │      ▼                │
          │    │    ┌──────────────────────────────────┐    │
          │    │    │   サンドボックス コンテナ            │    │
          │    │    │                                  │    │
          │    │    │   Claude Code (AI エージェント)    │    │
          │    │    │     ├─ ファイル読み書き             │    │
          │    │    │     ├─ コマンド実行                │    │
          │    │    │     ├─ Git操作                    │    │
          │    │    │     └─ Web検索                    │    │

               │                                       │
               │   ブランチ: claude/...                   │
               └───────────────────────────────────────┘
```

## ポイント

- ブラウザからAnthropicクラウド上のサンドボックスコンテナに接続（HTTPS/WebSocket）
- Claude Codeはサンドボックス内でファイル操作・Git操作・Web検索を実行
- 成果物はGitHubリポジトリにpush → ローカル端末からgit pullで取得
- ローカルファイルへの直接アクセスは不可
