東京発・新幹線利用、京都駅周辺泊。2026年7月に実際に歩いた行程をベースにしたモデルケース。
猛暑期の実施だったため、「朝イチに屋外・午後は屋内」の配分と、鉄道沿線での一筆書き動線が特徴。

## 全体サマリー

| 日 | エリア | 主な行程 |
|---|---|---|
| Day 1 | 洛中・北 | 移動 → 二条城 → 金閣寺 |
| Day 2 | 東山 | 清水寺 → 祇園 → 漢字ミュージアム → 南禅寺 → 哲学の道 → 銀閣寺 → 鴨川川床 |
| Day 3 | 伏見・宇治 | 伏見稲荷 → 宇治（平等院・茶づな）→ 三十三間堂 |
| Day 4 | 嵐山・太秦 | 竹林 → 天龍寺 → 渡月橋 → 太秦映画村 → 帰京 |

## Day 1: 移動 ＋ 二条城・金閣寺

| 時間 | 行程 |
|---|---|
| 午前 | 東京駅発 のぞみ → 京都着、ホテルに荷物を預ける |
| 昼前 | **二条城**（国宝・二の丸御殿。2026年は寛永行幸四百年祭） |
| 昼 | **蕎麦たつ市**（二条城近くの町家蕎麦。鴨せいろ・鴨なんばんが絶品） |
| 午後 | **金閣寺**（金色の舎利殿は圧巻。拝観受付は16:30頃まで） |
| 夕方 | ホテルチェックイン。翌朝に備えて早めに休む |

> **教訓**: 人気店の昼食は待ち時間で午後が押しやすい。この日も待ちが長引き、予定していた北野天満宮を断念。昼は開店直後を狙うか、待ちが出たら午後の行程を1つ削る判断を。

## Day 2: 東山縦断 ── 清水寺から銀閣寺まで

| 時間 | 行程 |
|---|---|
| 07:30 | **清水寺** 朝イチ参拝（開門6:00。8時台までは人が少なく涼しい） |
| 09:00 | 二年坂・三年坂を散策 |
| 10:00 | **八坂庚申堂**（くくり猿）→ **八坂神社** |
| 11:00 | **漢字ミュージアム**（八坂神社の目の前。冷房の効いた体験型施設で、暑い時間帯の避難先としても優秀） |
| 13:00 | 昼食：**南禅寺 順正**（名物の湯豆腐・引き上げ湯葉。庭園も見事） |
| 14:30 | 南禅寺側から**哲学の道**を北上 |
| 夕方 | **銀閣寺**（東山慈照寺） |
| 夜 | バスで祇園へ戻り、鴨川沿いを散歩しながら店探し → **京いざかや くぼ田**の川床で夕食（夏の鱧の湯引き・天ぷらが絶品） |

> **ポイント**: 「朝の清水寺 → 昼の屋内（漢字ミュージアム・順正）→ 夕方の銀閣寺 → 夜の川床」と、暑さのピークをうまく外した理想的な配分になった。

## Day 3: 伏見・宇治 ── 京阪沿線の一筆書き

| 時間 | 行程 |
|---|---|
| 朝 | **伏見稲荷大社** 朝イチで千本鳥居へ（混雑と暑さを避けるなら8時前着が理想） |
| 午前 | 参道で土産＆名物：**ちいかわもぐもぐ本舗**、**日野家**のスズメの串焼き |
| 昼過ぎ | 京阪で宇治へ → **平等院鳳凰堂** |
| 午後 | **増田茶舗**で宇治抹茶ラテ → **茶づな**（お茶と宇治のまち歴史公園）で涼む ※抹茶挽きなどの体験は時間指定制のため、やりたい場合は事前予約推奨 |
| 15:40頃 | 京阪宇治駅から乗車 → 七条駅下車 |
| 16:15 | **三十三間堂**（受付16:30まで。堂内は日差しがなく夕方でも快適） |
| 夕方 | 京都駅で土産購入 → 夕食 |

> **ポイント**: 伏見稲荷・宇治・三十三間堂はすべて京阪沿線。帰り道に七条で途中下車するだけで三十三間堂を消化でき、最終日の自由度が大きく上がった。猛暑日は午後を茶房・ミュージアムなど屋内で組むのが正解。

## Day 4: 嵐山・太秦 ＋ 帰京

| 時間 | 行程 |
|---|---|
| 08:00 | チェックアウト、荷物は京都駅コインロッカーへ |
| 08:45 | **竹林の小径**（朝の涼しく空いているうちに）→ そのまま**天龍寺 北門**から入り曹源池庭園を拝観、正門から出る |
| 昼前 | **渡月橋**を渡り、**峯嵐堂 渡月橋本店**でできたてわらび餅 |
| 昼 | **嵯峨とうふ 稲 本店**（天龍寺正門前）で湯葉うどん |
| 午後 | 嵐電で**東映太秦映画村**へ（2026年3月リニューアル。屋内アトラクション多めで午後の暑さ対策にも◎）。たっぷり遊ぶ |
| 夕方 | 京都駅へ → 駅弁を購入 |
| 19:00頃 | のぞみ乗車 → 21:15頃 東京着 |

> **ポイント**: 竹林 → 天龍寺北門 → 庭園 → 正門 → 渡月橋は、坂を戻らずに歩ける最効率ルート。嵐山と映画村は嵐電1本でつながる。最終日の新幹線を19時台にしたことで、映画村を駆け足にせず満喫できた。

## この旅から得た夏の京都Tips

1. **朝イチ屋外・午後屋内**の配分が夏の京都攻略の基本。漢字ミュージアム、茶づな、太秦映画村は冷房の効いた「避難先」としても機能する
2. **人気飲食店の待ち時間はスケジュールリスク**。開店直後を狙うか、押したら後続の行程を削る覚悟を
3. **京阪沿線（伏見稲荷〜宇治〜七条）は一筆書きの黄金ルート**。三十三間堂の「帰りに途中下車」は特におすすめ
4. **行けなかった場所は翌日以降に無理に押し込まない**。今回は北野天満宮・東寺を潔くスキップし、代わりに嵐山を回収して満足度が上がった
5. **帰りの新幹線は遅めに設定**しておくと最終日の自由度が段違い。駅弁で車内夕食にすれば時間を有効活用できる
6. 体験系施設（茶づなの抹茶挽きなど）は**時間指定制が多いので事前予約**を
