## 🎯 結論：Claude Max は Web 契約で月2,100〜4,300円安い

**Claude Max を使うなら、iOS（App Store）ではなく [claude.com](https://claude.com) から直接契約してください。** 機能はまったく同じなのに、契約経路を変えるだけで毎月これだけ違います（2026年7月・$1≈¥162.4・税込）。

| プラン | Web（claude.com）実質 | iOS（App Store） | **月の節約額** | **年の節約額** |
|:---|:---|:---|:---:|:---:|
| **Max 5x** | 約 **¥17,900**（$100） | **¥20,000** | **約 ¥2,100 安い** | **約 ¥25,200** |
| **Max 20x** | 約 **¥35,700**（$200） | **¥40,000** | **約 ¥4,300 安い** | **約 ¥51,600** |

> 同じ Claude Max を、iOS で契約すると **年に最大 約5.2万円**（Max 20x）よけいに払うことになります。差額の正体は Apple の手数料で、あなたが受け取るサービスは1ミリも変わりません。
>
> **【2026年7月の変化】** 6月版から **Apple が日本の App Store 円価格を引き下げ**（Max 5x ¥21,400→**¥20,000**、Max 20x ¥42,400→**¥40,000**）、加えて **円安（¥160→¥162.4）** で Web 側の円換算が上がったため、**Web 有利の差額は月¥3,800/¥7,200 → 月¥2,100/¥4,300 に縮小**しました。ただし **USD の +25% 構造（Apple 手数料の完全転嫁）は不変**で、Web が安い結論は変わりません。

**いま iOS で Max を契約している人も、Web に切り替えるだけでこの差額を取り戻せます**（手順は後半で解説）。

---

## 1. なぜここまで違うのか — 差はまるごと「Apple 手数料」

Apple の App Store でサブスクを契約すると、売上の **30%** が Apple のコミッションとして引かれます。Anthropic はこの手数料を **プランごとに違う扱い**にしています。

| プラン | Apple 手数料の扱い | iOS 価格への影響 |
|:---|:---|:---:|
| Pro 月額 | **Anthropic が吸収**（同額提供） | 影響なし |
| Pro 年契約 | 一部転嫁 | +7.5% |
| **Max 5x** | **完全転嫁** | **+25%** |
| **Max 20x** | **完全転嫁** | **+25%** |

Pro（月額 $20）のような低価格帯は Anthropic が手数料を肩代わりできますが、**Max は金額が大きく吸収しきれないため、Apple 手数料がそのまま価格に上乗せ**されます。つまり Max の iOS 価格に含まれる「割高分」は、Anthropic でもあなたでもなく **Apple に渡るだけ**のお金です。

---

## 2. 米ドル価格でも +25%（為替に関係なく Web が安い）

円換算前の素のドル価格で見ても、Max は iOS が +25% です。**為替がどう動こうと、この +25% という構造は変わりません。**

| プラン | Web（claude.com）月額 | iOS App Store 月額 | 差額 | 割高率 |
|:---|:---|:---|:---:|:---:|
| **Max 5x** | **$100** | $124.99 | +$24.99 | **+25%** |
| **Max 20x** | **$200** | $249.99 | +$49.99 | **+25%** |

- iOS の **Max には年契約が存在しない**（月額のみ）。長期割引も Web 限定です。
- Pro は米国だと Web / iOS とも月 $20 で同額ですが、**Max は構造的に iOS が +25%**。
- 円安（$1≈¥162.4）の今は、ここに **Web 側の消費税10%** も乗るため差額は縮みますが、それでもなお **Web が安い**という結論は揺らぎません（Apple の円値下げで6月より差は縮小）。

> 円高に振れれば Web 実質コストはさらに下がり、差はもっと開きます。**Max に限っては、為替がどちらに動いても Web 有利**です。

---

## 3. 年間でいくら損するか — 「気づかず払い続ける」のが一番もったいない

差額は月単位だと「数千円」ですが、サブスクは**毎月自動で更新**されます。1年放置すると：

| プラン | 月の差額 | **1年** | **2年** | **3年** |
|:---|:---:|:---:|:---:|:---:|
| Max 5x | 約 ¥2,100 | **約 ¥25,200** | 約 ¥50,400 | 約 ¥75,600 |
| Max 20x | 約 ¥4,300 | **約 ¥51,600** | 約 ¥103,200 | 約 ¥154,800 |

Max 20x を3年間 iOS で払い続けると、**Web 契約なら払わずに済んだ約15万円**を Apple 手数料として上乗せしていた計算になります。サービス内容は同じなのに、です。

**契約経路は一度見直せば、あとはずっと効く節約**です。最初の数分の手間で、毎月の固定費がそのまま下がります。

---

## 4. 機能は完全に同じ — 安いほうを選ばない理由がない

「安いほうは何か制限があるのでは？」と不安になるかもしれませんが、**Web 契約と iOS 契約に機能差はありません**。

| 観点 | Web 直接契約 | iOS App Store 経由 |
|:---|:---|:---|
| 使える機能・モデル | **同一** | **同一**（同じプランなら同じ） |
| 利用できる端末 | Web / iOS / Desktop すべて | Web / iOS / Desktop すべて |
| 領収書 | **Anthropic 名義**（インボイス対応可） | Apple 名義のみ |
| 法人経費・インボイス | ◎ | △（Apple 名義で経費処理に難） |
| 解約・プラン変更 | claude.com 設定画面 | Apple ID の購読管理 |
| 支払い手段 | クレジットカード等 | Apple Pay / Apple 課金 |

**Web で契約しても、iOS アプリ・Desktop アプリで同じアカウントにログインしてそのまま使えます**。つまり「Web で契約 → ふだんは iOS アプリで使う」が、機能を一切犠牲にせず一番安く済む構成です。法人で経費精算するなら、Anthropic 名義の領収書（インボイス対応）が出る Web 一択でもあります。

---

## 5. Web（claude.com）での契約手順

これから新規で Max を契約する人は、最初から Web で契約すれば差額を払わずに済みます。

1. PC やスマホのブラウザで [claude.com](https://claude.com) を開く
2. 既存の Claude アカウントでログイン（なければ新規登録）
3. 左下のアカウントメニュー →「**Upgrade / プランをアップグレード**」
4. **Max 5x（$100/月）** または **Max 20x（$200/月）** を選択
5. クレジットカード等で支払い手続き
6. 契約後、**iOS / Desktop アプリで同じアカウントにログイン**すれば、アプリでも Max として使える

> ポイント：**契約は Web、利用はどの端末でもOK**。iOS アプリを使い続けたい人も、契約だけ Web にすれば差額を払わずに済みます。

---

## 6. すでに iOS で契約している人の乗り換え手順

「もう iOS で Max を契約してしまった」場合でも、切り替えれば**今後の差額**を止められます。重要なのは、**Web と iOS は契約の直接移管ができない**点です（一度解約 → 新規契約の流れになります）。

1. **iOS 側で現在の購読を解約**
   - iPhone の「設定」→ 自分の名前 →「サブスクリプション」→ Claude →「サブスクリプションをキャンセル」
   - ⚠️ **解約しても、すでに支払った期間の終了日までは Max を使えます**（即停止ではない）
2. **現在の請求期間が終わるのを待つ**（期間内に二重課金しないため）
3. 期間終了後（または終了間際）に、**[claude.com](https://claude.com) で Web 契約**（上記「5. Web での契約手順」）
4. iOS / Desktop アプリは**同じアカウントでログインし直す**だけ。アプリの再インストールは不要

> タイミングのコツ：iOS の請求期間の**終了日近くに解約予約**し、終了後に Web 契約へ。これで「iOS と Web の二重払い」を避けつつ、空白期間も最小化できます。

---

## 7. 注意点

- **移管は不可・解約→再契約が必要**：Web ⇆ iOS は契約を直接引き継げません。乗り換えは必ず「iOS 解約 → Web 新規契約」の順で。
- **返金ポリシーが異なる**：iOS は Apple のポリシー（部分返金は条件付き）、Web は Anthropic のポリシーが適用されます。
- **価格・為替は変動する**：本記事は 2026年7月・$1≈¥162.4 時点。**USD 価格は6月から不変**（Web $100/$200・iOS $124.99/$249.99）だが、**Apple が日本の円価格を引き下げ**（¥20,000/¥40,000）、**円安**も進んだため差額は6月より縮小した。**+25% という USD 構造ゆえ Web 有利は不変**だが、実額（円）は Apple の円ティア改定・為替で変わる。最新価格は claude.com と App Store で確認を。
- **Pro は事情が違う**：低価格帯の **Pro は円安だと iOS と Web がほぼ互角**（むしろ iOS が微安なことも）。本記事は **Max 限定の結論**です。Pro も含めた全体比較は関連記事を参照してください。

## まとめ

- **Claude Max は Web（claude.com）契約が月 約¥2,100〜¥4,300 安い**（Max 5x / Max 20x、2026年7月・$1≈¥162.4）。
- 年間では **約¥25,200〜¥51,600** の差（6月版の¥45,600〜¥86,400 から縮小＝Apple の円値下げ＋円安）。差の正体は **Apple 手数料（Max は +25% 完全転嫁・USD では不変）**で、サービス内容は完全に同一。
- **為替がどう動いても Max は Web 有利**（米ドルでも +25%）。
- すでに iOS 契約中でも、**iOS 解約 → Web 再契約**で今後の差額を止められる。契約は Web、利用は全端末でOK。

**Max を検討しているなら、最初から claude.com で契約する**——それだけで毎月の固定費がそのまま下がります。
